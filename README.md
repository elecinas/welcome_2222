# Barcelona Fake — Cómic Interactivo

Un cómic web interactivo con estética inspirada en Arcane, ambientado en una Barcelona idealizada del año 2025.

## Características
- Estética "Dark Mode" con acentos dorados.
- Diseño minimalista y ligero (sin frameworks).
- Rutas narrativas basadas en decisiones del usuario.

## Cómo usar
Simplemente abre el archivo `index.html` en cualquier navegador moderno para comenzar la experiencia.
